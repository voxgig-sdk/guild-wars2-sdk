# guild-wars2-mcp

MCP server exposing the GuildWars2 SDK as tools, built on the
[official Go MCP SDK](https://github.com/modelcontextprotocol/go-sdk)
and the sibling Go SDK at `../go`.

## Tools

| Tool | Args | Returns |
|------|------|---------|
| `guild-wars2_list` | `entity`, optional `query` map | First page of records as JSON |
| `guild-wars2_load` | `entity`, `query` (e.g. `{id:N}`) | Single record as JSON |

JSON schemas for both tools are emitted by the SDK from the `Args`
struct's `json` / `jsonschema` tags — no schema is hand-written.

## Entities

achievement | authenticated | daily_reward | game_mechanic | guild | guild_authenticated | home_instance | item | map | map_information | miscellaneous | story | structured_pv_p | trading_post | world_vs_world

## Build

```sh
go build -o guild-wars2-mcp ./...
```

## Run

```sh
# stdio transport — for spawnable agent installs (default).
./guild-wars2-mcp

# streamable HTTP transport — share one running server between agents.
./guild-wars2-mcp -transport http -addr :8080
```

## Install for Claude Code

```sh
claude mcp add --scope user guild-wars2 \
  -- /absolute/path/to/guild-wars2-mcp -transport stdio
```

After install, restart Claude Code; the `guild-wars2_list` and
`guild-wars2_load` tools become available in new sessions.

## Smoke test via HTTP

```sh
./guild-wars2-mcp -transport http -addr :18080 &

# initialize, grab the session id
curl -sN -X POST http://localhost:18080 \
  -H 'Content-Type: application/json' \
  -H 'Accept: application/json, text/event-stream' \
  -D headers \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-06-18","capabilities":{},"clientInfo":{"name":"smoke","version":"0"}}}'

SESSION=$(awk '/Mcp-Session-Id/ {print $2}' headers | tr -d '\r')

curl -sN -X POST http://localhost:18080 \
  -H 'Content-Type: application/json' \
  -H 'Accept: application/json, text/event-stream' \
  -H "Mcp-Session-Id: $SESSION" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"guild-wars2_load","arguments":{"entity":"achievement","query":{"id":1}}}}'
```
